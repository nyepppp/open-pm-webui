# Quickstart: PM 工作台重构验证指南

**Feature**: PM 工作台重构 — 模块化布局与差异化编辑器
**Date**: 2026-06-28
**Version**: 1.0.0

---

## 前提条件

- Open WebUI 已安装并运行
- PM 工作台模块已启用
- 至少创建一个测试项目

---

## 验证场景 1：侧边栏分类导航

**目标**: 验证侧边栏按 4 大分类组织模块

### 步骤

1. 登录 Open WebUI，进入 PM 工作台
2. 观察左侧侧边栏，确认显示 4 个分类：
   - 规划类（PRD文档、需求池、竞品分析、路线图）
   - 设计类（参数清单、测试用例）
   - 执行类（风险问题、会议纪要）
   - 复盘类（验收复盘、FAQ/培训）
3. 点击"规划类"，确认展开显示 4 个子菜单
4. 点击"PRD文档"，确认主区域显示富文本编辑器
5. 确认 URL 更新为 `/pm/prd`

### 预期结果

- 侧边栏分类可展开/折叠
- 选中模块高亮显示
- 主区域正确加载对应编辑器

---

## 验证场景 2：差异化编辑器

**目标**: 验证不同模块使用不同编辑器

### 步骤

1. 进入"PRD文档"模块，确认显示富文本编辑器（支持 Markdown、图片、表格）
2. 进入"参数清单"模块，确认显示结构化表单（字段名、类型、必填等）
3. 进入"风险问题"模块，确认显示混合编辑器（表单字段 + 富文本描述）
4. 进入"路线图"模块，确认显示思维导图视图

### 预期结果

- PRD：富文本编辑器，支持标题、版本、状态下拉框
- 参数清单：结构化表单，类型下拉框（string/number/boolean/object/array）
- 风险：混合编辑器，风险等级下拉框 + 概率滑块 + 富文本区
- 路线图：思维导图，可拖拽节点

## 验证场景 3：版本控制

**目标**: 验证版本切换功能

### 步骤

1. 在 PRD 文档模块创建一条内容
2. 点击顶部"创建版本"按钮，输入版本号 "v1.0"
3. 修改 PRD 内容
4. 再次创建版本 "v1.1"
5. 点击顶部版本切换下拉框，选择 "v1.0"
6. 确认内容回退到 v1.0 状态

### 预期结果

- 版本切换后数据正确刷新
- 版本历史列表显示所有版本
- 版本对比显示差异

## 验证场景 4：数据关联

**目标**: 验证模块间关联建立

### 步骤

1. 在"参数清单"模块创建一条参数
2. 在"来源PRD"字段下拉框中选择已有的 PRD 文档
3. 保存参数
4. 进入该 PRD 文档，查看关联信息
5. 确认显示关联的参数清单条目

### 预期结果

- 关联字段下拉框正确显示可选目标
- 关联建立后双向可见
- 删除 PRD 后显示"[已删除的PRD]"

## 验证场景 5：Agent 智能辅助

**目标**: 验证 Agent 分析建议功能

### 步骤

1. 在 PRD 文档模块编写一段不完整的内容
2. 点击"AI分析"按钮
3. 等待 Agent 分析完成
4. 查看建议列表：缺失章节、逻辑不完整、可补充细节
5. 点击"采纳"按钮，确认建议应用到文档

### 预期结果

- Agent 返回 actionable 建议
- 建议标记为"待确认"状态
- 用户确认后才生效

---

## 验证场景 6：数据迁移

**目标**: 验证旧数据自动迁移

### 步骤

1. 准备一个包含旧格式数据（`{text: string}`）的项目
2. 首次访问该项目的"需求池"模块
3. 确认显示迁移摘要面板
4. 查看迁移条目数、字段映射情况
5. 修正异常项（如标题过长）
6. 点击"确认迁移"按钮

### 预期结果

- 旧数据自动转换为新格式
- 默认值正确填充（title 从 text 提取，priority 默认 P2）
- 迁移失败条目标记为"待处理"

## 验证场景 7：并发编辑冲突

**目标**: 验证乐观锁冲突处理

### 步骤

1. 在浏览器 A 中打开某条 PRD 文档
2. 在浏览器 B 中打开同一条 PRD 文档
3. 在浏览器 A 中修改内容并保存
4. 在浏览器 B 中修改内容并保存
5. 确认浏览器 B 显示冲突提示

### 预期结果

- 显示"该条目已被其他用户修改"提示
- 提供"覆盖/合并/放弃"选项
- 选择"合并"时展示差异对比

## 常见问题排查

### 问题 1：富文本编辑器无法加载

**现象**: 模块显示空白或报错
**解决**: 检查网络连接，确认 TipTap 库已正确加载。如仍失败，系统会自动降级为纯文本 textarea。

### 问题 2：版本切换后数据未刷新

**现象**: 切换版本后显示旧数据
**解决**: 检查网络请求是否成功，确认版本 ID 正确传递。如问题持续，尝试刷新页面。

### 问题 3：Agent 分析超时

**现象**: 点击"AI分析"后长时间无响应
**解决**: 检查 API Key 配置，确认 AI 服务可用。如超时，显示"分析超时，请重试"提示。
