# PM 工作台 — 模块化布局与差异化编辑器

> 重构 PM 工作台，实现分类导航、差异化编辑、数据互通和 AI 智能辅助

## 功能概览

### 1. 模块化侧边栏导航

左侧分类侧边栏按 4 大业务分类组织模块：

| 分类 | 模块 |
|------|------|
| 🗺️ 规划 | PRD 文档、需求池、竞品分析、路线图 |
| 🎨 设计 | 参数清单、测试用例 |
| ⚡ 执行 | 风险问题、会议纪要 |
| 📊 复盘 | 验收复盘、FAQ/培训、产品架构 |

- 侧边栏支持展开/折叠，移动端自动收起
- 键盘导航：上下箭头在分类和模块间移动，Enter/空格展开或导航
- 当前选中模块高亮显示

### 2. 差异化编辑器

每个模块根据业务特点使用不同编辑器：

| 编辑器类型 | 适用模块 | 特点 |
|------------|---------|------|
| 富文本编辑器 | PRD、竞品、会议、验收、FAQ | 基于 TipTap，支持标题/列表/代码块/Markdown |
| 结构化表单 | 需求、参数、测试用例 | zod 验证，字段类型包括文本/下拉/日期/JSON |
| 混合编辑器 | 风险 | 上半部分表单 + 下半部分富文本 |
| 思维导图 | 路线图、产品架构 | 基于 @xyflow/svelte，可拖拽/缩放/自定义节点 |

**性能优化**：
- 富文本编辑器：50,000 词大文档懒加载、防抖更新、大文档禁用历史记录
- 思维导图：200+ 节点时自动启用视口过滤，仅渲染可见区域节点
- 列表视图：100+ 条目时自动切换虚拟滚动

### 3. 版本控制

- 项目级版本快照 + 模块级增量标记
- 版本创建：点击顶部版本号 → "创建新版本"
- 版本切换：下拉选择，1 秒内数据刷新
- 版本对比：显示新增/修改/删除的结构化差异
- 版本回滚：支持整项目回滚和单模块回滚
- 未保存变更提示：切换版本前提醒保存

### 4. 数据关联与追溯

- 模块间关联：参数 → PRD、测试用例 → 需求/参数
- 双向可见：在 PRD 查看关联的参数和测试用例
- 影响分析：展示上游/下游依赖链路
- 追溯图：可视化实体间的关联关系
- 删除引用处理：已删除实体显示 "[已删除的PRD]"

### 5. AI 智能辅助

- 手动触发：每个模块的 "AI 分析" 按钮
- 自动触发：保存时自动分析（可在设置中配置/关闭）
- 分析类型：完整性检查、风险识别、关联建议
- 人工确认：AI 建议标记为"待确认"，用户需手动采纳/拒绝
- 降级处理：AI 服务不可用时显示友好提示，所有模块仍可手动编辑

### 6. 数据迁移

- 自动检测旧格式 `{text: string}` 数据
- 迁移面板显示摘要和字段映射
- 默认值自动填充（title 从 text 提取，priority 默认 P2）
- 迁移失败条目标记为"待处理"

## 错误处理与降级

| 场景 | 处理方式 |
|------|---------|
| 富文本编辑器加载失败 | 自动降级为纯文本 textarea，显示警告提示 |
| AI 服务不可用 | 显示友好错误信息和重试按钮，手动编辑不受影响 |
| AI 请求超时/限流 | 分类错误信息（API Key 未配置/频率超限/服务维护），提供解决建议 |
| 网络中断 | 所有已加载功能正常使用，仅影响需要 API 的操作 |

## 无障碍访问

- 侧边栏：键盘导航（方向键/Enter/Escape/Home/End）
- 所有交互元素：ARIA 标签
- 版本选择器：对话框模式、搜索框标签、选项角色
- 列表视图：列表项角色和选中状态

## 技术栈

| 技术 | 用途 |
|------|------|
| SvelteKit | 前端框架 |
| TipTap | 富文本编辑器 |
| @xyflow/svelte | 思维导图 |
| zod | 表单验证 |
