# PM工作台溯源下拉列表新增版本信息展示

## Goal

在PM工作台所有溯源（Trace）相关的下拉选择组件中，除了显示溯源名称外，增加显示该溯源所属的版本信息，帮助用户区分同名不同版本的溯源。

## Requirements

1. **展示格式**：下拉选项中同时显示溯源名称和版本号，格式统一为 `{溯源名称} (v{版本号})`
2. **涉及范围**：PM工作台中所有使用溯源下拉选择器的页面/组件（包括但不限于溯源选择、溯源筛选等）
3. **数据要求**：下拉选项数据需包含 `version` 或 `version_id` 字段
4. **搜索兼容**：下拉搜索/过滤功能需支持基于名称或版本号的搜索
5. **回显兼容**：已选择的溯源在输入框中回显时，同样展示版本信息

## Acceptance Criteria

- [ ] 所有溯源下拉选择组件的选项中同时展示溯源名称和版本号
- [ ] 版本信息展示格式统一、清晰
- [ ] 下拉搜索/过滤功能正常工作（基于名称或版本号）
- [ ] 选择后的数据回显和表单提交逻辑不受影响
- [ ] 代码通过前端类型检查（TypeScript）
