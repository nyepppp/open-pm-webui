# 功能: 参数模块调整(输入/默认值/主次)

## Goal

对参数模块进行三项调整：下拉支持手动输入、新增默认值参数、调整主次关系（模块和功能为主，参数为次）。

## Requirements

- 批注 #11: "选择模块"下拉需要同时支持手动输入
  - 选择器: `select[class="w-full text-sm px-3 py-2 bg-gray-50 dark:bg-gray-850"]`
- 批注 #12: "选择功能"下拉需要同时支持手动输入
  - 选择器: 同上（第二个 select）
- 批注 #13: 新增"是否有默认值"的参数
- 批注 #14: 有的功能不一定要有参数，模块和功能应是主，参数是次（主次混淆了）
- 页面路径: `/pm/{projectId}/parameter`

## Acceptance Criteria

- [ ] "选择模块"下拉框支持手动输入（combobox 模式）
- [ ] "选择功能"下拉框支持手动输入（combobox 模式）
- [ ] 参数表单新增"是否有默认值"字段
- [ ] 参数区域在 UI 上明确为次要信息，模块和功能为主要信息
- [ ] 功能可以不关联参数（参数为可选）
