# 修复 Roadmap 页面 SVG 点击和日程同步功能

## Goal

修复 `/pm/:projectId/roadmap` 页面中的两个功能问题：
1. SVG 日历同步按钮点击无响应
2. 日程同步功能的错误处理和用户体验优化

## Requirements

### P0 - 必须修复
1. **SVG 点击修复**: 移除导致点击事件被吞掉的 `pointer-events-none` 类
2. **错误提示增强**: 确保所有错误路径都有明确的用户提示（toast）

### P1 - 建议优化
3. **同步状态可视化**: 在按钮上区分已同步/未同步状态
4. **日期处理健壮性**: 增强日期解析，处理边界情况

## Acceptance Criteria

- [ ] 点击"同步到日程"按钮能正常触发同步流程
- [ ] 未登录时给出明确提示
- [ ] 没有可用日历时给出明确提示
- [ ] 同步成功后有成功提示
- [ ] 同步失败后有错误提示
- [ ] 已同步的条目再次同步时提示是否覆盖
- [ ] 同步按钮的视觉状态能反映同步状态
