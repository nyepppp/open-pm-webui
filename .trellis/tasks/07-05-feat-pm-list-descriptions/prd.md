# 功能: PM列表新增描述字段

## Goal

在 PM 项目列表卡片中新增创建时间、当前版本、上次修改时间三个描述字段。

## Requirements

- 批注 #4: PM 列表项新增描述 — 创建时间、当前版本、上次修改时间
- 选择器: `a[class="flex cursor-pointer w-full px-3.5 py-1.5 border"]`（列表卡片）
- 页面路径: `/pm`

## Acceptance Criteria

- [ ] PM 项目列表卡片显示创建时间
- [ ] PM 项目列表卡片显示当前版本号
- [ ] PM 项目列表卡片显示上次修改时间
- [ ] 新增字段布局不破坏现有卡片样式
